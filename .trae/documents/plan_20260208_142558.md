# 修复前端样式和接口404问题

## 问题分析

### 1. 样式问题
- **现象**: 登录页面布局不够美观，卡片宽度和内边距需要调整
- **原因**: 登录页面的样式配置不够优化
- **影响**: 用户体验不佳，界面不够专业

### 2. 接口404问题
- **现象**: 前端发送的API请求 `http://localhost:5173/api/auth/login` 返回404错误
- **原因**: Vite配置文件中缺少API代理设置，前端直接向自己的服务器请求API
- **影响**: 登录功能无法使用，系统无法正常认证

## 解决方案

### 1. 修复样式问题
- **文件**: `/frontend/src/pages/Login.tsx`
- **修改**: 优化登录卡片样式，调整宽度、内边距和表单元素样式
- **代码变更**:
  ```tsx
  // 调整卡片样式
  <Card title="劳动仲裁调解系统" style={{ 
    width: 500, 
    borderRadius: 8, 
    padding: '24px',
    boxShadow: '0 4px 12px rgba(0,0,0,0.15)'
  }}>

  // 优化表单样式
  <Form
    form={form}
    name="login"
    onFinish={onFinish}
    layout="vertical"
    size="middle"
  >
  ```

### 2. 修复接口404问题
- **文件**: `/frontend/vite.config.ts`
- **修改**: 添加API代理配置，将/api请求转发到后端服务器
- **代码变更**:
  ```ts
  import { defineConfig } from 'vite'
  import react from '@vitejs/plugin-react'

  export default defineConfig({
    plugins: [react()],
    server: {
      proxy: {
        '/api': {
          target: 'http://localhost:5002',
          changeOrigin: true,
          secure: false
        }
      }
    }
  })
  ```
